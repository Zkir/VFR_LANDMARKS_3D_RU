УСТАНОВКА
------------
Данный сценарий в scenery_packs.ini  должен находиться 
ниже специфических сценариев, но выше мешей.

Никакие дополнительные библиотеки не требуются.

Настоятельно рекомендуется использовать совместно с HD Mesh Scenery v4 от alpilotx.
оно заменяет рельеф и дает очень приличный автоген. 

http://www.alpilotx.net/downloads/x-plane-11-hd-mesh-scenery-v4/

НЕ рекомендуется использовать другие конвертации из OSM, они очень примитивные.

-------------------
